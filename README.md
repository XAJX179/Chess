## About

  Chess game with terminal ui and mouse click support,
  load game with FEN codes or start new!

## Install

> [!IMPORTANT]
> pre-requisite: ruby >= 3.2

```bash
gem install chess_tui
```

## uninstall

```bash
gem uninstall chess_tui
```
